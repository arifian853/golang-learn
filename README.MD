## How to Install Go

1. Download and install Go from: https://golang.org/dl/
2. Make sure Go is added to your PATH
3. Verify installation:

   ```bash
   go version
   ```

## How to Run the Code

Inside a chapter or project folder:

```bash
go run main.go
```

Or build the binary:

```bash
go build main.go
```
